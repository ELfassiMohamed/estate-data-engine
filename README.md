# Estate Data Engine

Asynchronous real estate data pipeline designed to scrape, process, and store real estate listings using **Playwright** and **PostgreSQL**.


##  Tech Stack

- **Language**: Python 3.10+
- **Automation**: [Playwright](https://playwright.dev/python/)
- **Database**: [PostgreSQL](https://www.postgresql.org/)
- **Driver**: [psycopg](https://www.psycopg.org/psycopg3/)
- **Environment**: [python-dotenv](https://github.com/theskumar/python-dotenv)
